## Run Locally

Clone the project

```bash
  git clone https://github.com/shubyaa/e_commerce_app
```

Go to the project directory

```bash
  cd e_commerce_app
```

Install dependencies

```bash
  flutter pub get
```

Start debugging

```bash
  flutter run
```

To generate APK file

```bash
  flutter build apk --release
```
